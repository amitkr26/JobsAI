# JobsAI / ElectroBridge

**ElectroBridge** — A one-stop aggregator for electronics & semiconductor opportunities in India and globally. Built with Next.js, Supabase, and Tailwind CSS.

## Getting Started

```bash
cd electrobridge
npm install
# Set up .env.local with Supabase credentials
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Tech Stack

- **Next.js 14** (App Router)
- **Supabase** (PostgreSQL)
- **Tailwind CSS**
- **Vercel** (deployment)

## Deploy

1. Push to GitHub
2. Import in [Vercel](https://vercel.com)
3. Add environment variables
4. Deploy
